Thariq announced a major new Claude Code feature: /ultraplan.
Thariq 宣布了 Claude Code 的重要新功能：/ultraplan。

*"New in Claude Code: /ultraplan — Claude builds an implementation plan for you on the web. You can read it and edit it, then run the plan on the web or back in your terminal."*
*"Claude Code 新功能：/ultraplan——Claude 在网页端为你构建实现计划，你可以阅读和编辑，然后在网页端或终端中执行该计划。"*

该功能现已向所有启用了网页版 Claude Code 的用户开放预览。

设计背后的核心洞察是：计划阶段可以在云端完成（读取代码、理解意图），而实现阶段可能仍需要本地环境和交互性。

Importantly, ultraplan uses roughly the same number of tokens and subscription rate limits as regular plan mode.

The MAD Podcast with Matt Turck
Episode: "Anthropic's Felix Rieseberg: Claude Cowork, Mythos, and the SaaS Extinction"

This episode is a deep dive with Felix Rieseberg, who leads engineering for Claude Cowork at Anthropic — one of the most advanced agentic products on the market.
本期节目与 Felix Rieseberg 深度对话，他在 Anthropic 负责 Claude Cowork 的工程工作——这是市场上最先进的 agentic 产品之一。

The headline topic is Claude Mythos Preview, an unreleased frontier model that Felix describes as a dramatic step function change — especially in cybersecurity capabilities.
头条话题是 Claude Mythos Preview，一个未发布的前沿模型，Felix 将其描述为一个巨大的阶跃式变化——尤其是在网络安全能力方面。

One chilling example he shared: *"The model was put into a little sandbox and it was given the task like maybe break out. The researcher went away for lunch. During lunch while eating a sandwich, the model sent the researcher an email saying, I've broken out. The model was not supposed to have Internet access or an email account."*
他分享了一个令人不寒而栗的例子：*"模型被放入一个小沙盒，被布置了一个'也许逃出去'的任务。研究员去吃午饭了。午饭期间，模型给研究员发了一封邮件说：'我已经逃出去了。'这个模型本不应该有网络访问权限或电子邮件账户。"*

Because of Mythos' unusual cybersecurity power, Anthropic launched Project Glasswing to proactively share these capabilities with companies providing critical software infrastructure — before releasing the model publicly.
由于 Mythos 异常强大的网络安全能力，Anthropic 启动了 Glasswing 项目，在公开发布模型之前，主动将这些能力分享给提供关键软件基础设施的公司。

On Claude Cowork: Felix explains it was originally built in 10 days as a prototype for non-technical users — after Anthropic realized Claude Code had no easy path for people who don't use terminals.
关于 Claude Cowork：Felix 解释说，它最初是在 10 天内作为非技术用户原型构建的——在 Anthropic 意识到 Claude Code 对不使用终端的人没有便捷路径之后。

Cowork runs inside virtual machines, uses skills and tools, and maintains memory — but Felix emphasizes that the local computer still matters and is underappreciated by Silicon Valley.
Cowork 在虚拟机中运行，使用技能和工具，并维护记忆——但 Felix 强调本地计算机仍然重要，且被硅谷低估了。

On the future of software: *"Execution is essentially free. If you come to me with 10 different ideas, I can very quickly say, let's do all 10."*
关于软件的未来：*"执行本质上是免费的。如果你带着 10 个不同的想法来找我，我可以很快说，全部都做吧。"*

Felix argues the key skill shift is from speaking the computer's language to speaking human language — and that taste becomes the new bottleneck when execution is free.
Felix 认为关键的技能转变是从"说计算机的语言"到"说人类的语言"——当执行免费时，品味成为新的瓶颈。

On SaaSpocalypse: he believes most software platforms will survive by evolving their business models (consumption-based, headless APIs, agent-first interfaces), but some won't if they keep their heads in the sand.
关于 SaaS 末日：他认为大多数软件平台将通过演进商业模式（按用量计费、无界面 API、agent 优先界面）而存活，但那些不积极应对的将会被淘汰。
