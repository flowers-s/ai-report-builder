### Andrej Karpathy (karpathy on X)

There is a growing gap in understanding of AI capability, and Karpathy believes it comes from two distinct user groups.
AI 能力的理解鸿沟正在扩大，Karpathy 认为这源于两类截然不同的用户群体。

The first group tried free or old ChatGPT and formed their views from hallucinations and fumbled queries.
第一类人使用的是免费或旧版 ChatGPT，他们对 AI 的印象来自幻觉和蹩脚的回答。

The second group pays for and uses state-of-the-art frontier agentic models (OpenAI Codex / Claude Code) in technical domains like programming and research.
第二类人付费使用前沿 Agent 模型（OpenAI Codex / Claude Code），应用于编程、数学、研究等技术领域。

*"This group of people is subject to the highest amount of 'AI Psychosis' because the recent improvements in these domains as of this year have been nothing short of staggering."*
*"这类人是'AI 妄想症'最严重的群体，因为今年这些领域的进步幅度令人瞠目结舌。"*

The key reason coding agents shine: verifiable reward functions (unit tests pass or fail) + massive b2b commercial value = all the focus and investment goes there.
编程 Agent 领先的关键：可验证的奖励函数（单元测试通过与否）+ 巨大的 B2B 商业价值 = 所有注意力和投资都集中于此。

He also noted the "OpenClaw moment" — when a large group of non-technical people for the first time experienced the latest agentic models, not just ChatGPT-the-website.
他还提到"OpenClaw 时刻"——当大批非技术用户首次亲身体验最新 Agent 模型，而非只是网页版 ChatGPT 时，一切都变了。

### Claude / Anthropic (claudeai on X)

Anthropic launched a major new feature in beta: the **Opus Advisor Tool** for the Messages API.
Anthropic 在 beta 版中发布了重大新功能：Messages API 的 **Opus Advisor Tool**（顾问工具）。

When a Sonnet or Haiku agent hits a hard decision mid-run, it can now "phone a friend" — consult Opus, get a plan, and continue, all within a single API request.
当 Sonnet 或 Haiku Agent 在运行途中遇到难题，现在可以"打电话求助"——咨询 Opus、获取方案，然后继续，整个过程在同一次 API 请求内完成。

In evals, Sonnet with an Opus advisor scored 2.7 percentage points higher on SWE-bench Multilingual than Sonnet alone, while costing 11.9% less per task.
在评估中，搭载 Opus 顾问的 Sonnet 在 SWE-bench 多语言测试中比单独使用 Sonnet 高出 2.7 个百分点，而每任务成本降低了 11.9%。

### Unsupervised Learning — Ep 84: OpenAI's Chief Scientist on Continual Learning Hype, RL Beyond Code, & Future Alignment Directions

This episode features Ako Paioki, Chief Scientist of OpenAI, in a wide-ranging conversation with host Jacob Efron (Redpoint Ventures).
本期播客邀请了 OpenAI 首席科学家 Ako Paioki，与主持人 Jacob Efron（Redpoint Ventures）进行了深度对话。

**On model progress and coding agents:**
**关于模型进展与编程 Agent：**

Ako confirmed that OpenAI now uses Codex for the majority of actual coding internally, calling it a clear signal that "something here is on track."
Ako 确认 OpenAI 内部现已将 Codex 用于大多数实际编程工作，称这是"一切正在正轨上"的明确信号。

He expects models to be able to work autonomously for a couple of days within months, with much higher quality artifacts.
他预计几个月内模型就能自主工作数天，并产出质量更高的成果。

**On reinforcement learning beyond code:**
**关于代码之外的强化学习：**

Ako believes RL will generalize to harder domains (medicine, law, finance), linking these to longer-horizon tasks where even well-specified problems become open-ended over time.
Ako 认为强化学习将推广到更难的领域（医疗、法律、金融），将其与更长期任务联系起来——即使是定义明确的问题，随着时间推移也会变得开放式。

*"I kind of believe these difficulties coincide, and they're very clearly the next frontier for how these systems develop."*
*"我认为这些困难是一致的，这显然是这些系统发展的下一个前沿。"*

**On continual learning "hype":**
**关于持续学习"炒作"：**

He pushed back on the framing, saying the entire GPT scaling excitement was always about continual/in-context learning.
他对这一框架提出异议，指出整个 GPT 扩展的兴奋点一直都是关于持续学习/上下文学习。

*"I definitely agree that continual learning is really the thing. It is what we're working towards."*
*"我完全同意，持续学习确实是关键。这正是我们正在努力的方向。"*

**On alignment and chain-of-thought monitoring:**
**关于对齐与思维链监控：**

Ako explained why OpenAI hides the chain of thought in reasoning models: because if you show it in the product, you eventually have to train it, which would undermine its value as an unsupervised interpretability signal.
Ako 解释了 OpenAI 隐藏推理模型思维链的原因：如果在产品中展示，最终就不得不训练它，从而破坏其作为无监督可解释性信号的价值。

He described alignment's long-term challenge as a generalization problem — what values does a much more capable model fall back on?
他将对齐的长期挑战描述为泛化问题——一个更强大的模型会回归到什么价值观？

He is optimistic about chain-of-thought monitoring as a tool for this, and excited about work on model scheming in collaboration with Anthropic and DeepMind.
他对思维链监控作为解决这一问题的工具持乐观态度，并对与 Anthropic 和 DeepMind 合作开展的模型行为研究感到兴奋。

**On AI for science:**
**关于 AI 科学应用：**

Recalling the FrontierMath challenge, Ako described the emotional experience of watching a model solve problems from his own PhD domain in about an hour — problems he'd have been proud to solve in a week.
回忆 FrontierMath 挑战赛时，Ako 描述了看着模型在约一小时内解决他博士研究领域问题时的复杂情绪——这些问题他自己可能需要一周才能解决。

*"Seeing that happen for math...definitely really increased my feeling of urgency."*
*"看到这发生在数学上……确实大大加强了我的紧迫感。"*

**On societal impact:**
**关于社会影响：**

He identified two underappreciated challenges: job displacement/wealth concentration requiring policymaker involvement, and the question of governance for organizations that are incredibly powerful yet made of very few people.
他指出了两个被低估的挑战：需要政策制定者介入的就业替代/财富集中问题，以及对极具影响力却由极少数人控制的组织的治理问题。
