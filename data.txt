**Latent Space — Extreme Harness Engineering for Token Billionaires: 1M LOC, 1B toks/day, 0% human code, 0% human review — Ryan Lopopolo, OpenAI Frontier & Symphony**

Ryan Lopopolo from OpenAI's Frontier team explains "Harness Engineering" — a radical methodology where AI agents do 100% of the coding and code review, with zero human-written or human-reviewed code in a 1M+ line codebase.
OpenAI Frontier 团队的 Ryan Lopopolo 详解"Harness Engineering"——一种激进的工程方法论：在超过 100 万行代码的代码库中，所有代码均由 AI Agent 编写和审查，零人工编写、零人工审查。

Over five months, his team shipped an internal beta product consuming over 1 billion tokens per day (~$2–3k/day), iterating through multiple Codex model generations.
五个月内，他的团队发布了一款内部 Beta 产品，每天消耗超过 10 亿 Token（约合每日 2000-3000 美元），并历经多代 Codex 模型的迭代。

A key lesson: when the agent fails, don't prompt it harder — instead ask "what capability, context, or structure is missing?" and build smaller building blocks.
核心经验：当 Agent 失败时，不要强迫它"再试一次"，而应追问"缺少什么能力、上下文或结构？"，然后拆解成更小的构建模块。

The resulting open-source project Symphony is a "ghost library" reference Elixir implementation that sets up a massive system of Codex agents.
最终产出的开源项目 Symphony 是一个"幽灵库"参考实现（Elixir），可搭建大规模 Codex Agent 协作体系。
